# 🌌 MidnightViolet

**System24 supercharged** — un tema inky dark con accenti viola raffinati, progettato per sessioni prolungate su Discord.

Basato su [System24](https://github.com/refact0r/system24) di refact0r, pesantemente personalizzato con una palette viola scuro, glow effects, animazioni custom e syntax highlighting dedicato.

---

## ✨ Features

- 🎨 **Palette Viola Raffinata** — sfondo quasi nero puro con accenti viola a saturazione controllata
- 🔤 **JetBrains Mono + Space Grotesk** — tipografia monospace professionale
- ✨ **Glow Effects** — bordi che brillano al hover su server, input, menzioni
- 🎬 **Micro-Animazioni** — ingresso messaggi, popout, placeholder lampeggiante
- 🖥️ **Syntax Highlighting Custom** — colori neon per code blocks
- 📜 **Scrollbar Sottile** — scrollbar neon minimal
- 🔔 **Badge & Notifiche** — glow effect sulle notifiche
- 🎵 **Spotify Custom Bar** — barra di riproduzione personalizzata
- ♿ **Focus Accessibilità** — outline viola sugli elementi focus

## 🚀 Installazione su Vesktop

### Metodo 1 — Importazione Online (Consigliato)

Questo metodo carica il tema direttamente da GitHub, così ricevi automaticamente gli aggiornamenti.

1. Apri **Vesktop**
2. Vai su **Settings** → **Vencord** → **Themes**
3. Nella sezione **Online Themes**, incolla questo link nella casella di testo:

```
https://raw.githubusercontent.com/L0orexist/MidnightVioletTheme/master/MidnightViolet.theme.css
```

4. Premi **Invio** per confermare
5. Il tema verrà applicato immediatamente! 🎉

---

### Metodo 2 — Installazione Locale (File CSS)

Se preferisci avere il file in locale:

1. **Scarica il file** [`MidnightViolet.theme.css`](https://github.com/L0orexist/MidnightVioletTheme/raw/master/MidnightViolet.theme.css)  
   *(Click destro → "Salva link con nome..." oppure scarica lo ZIP del repo)*
2. Apri **Vesktop**
3. Vai su **Settings** → **Vencord** → **Themes**
4. Clicca su **Open Themes Folder** (📁)
5. **Copia** il file `MidnightViolet.theme.css` nella cartella che si è aperta
6. Torna su Vesktop — il tema apparirà nella lista
7. **Attivalo** con il toggle ✅

> [!TIP]
> Con il metodo locale, per aggiornare il tema dovrai scaricare nuovamente il file.

## 🔧 Personalizzazione

Puoi modificare facilmente i colori e le opzioni editando le variabili nel file CSS:

```css
/* Cambia il font principale */
--font: 'JetBrains Mono';

/* Modifica il colore accento principale */
--accent-1: oklch(72% 0.13 295);

/* Regola la luminosità dello sfondo */
--bg-4: oklch(9% 0.008 290);
```

Per una lista completa delle variabili configurabili, consulta la [documentazione di System24](https://github.com/refact0r/system24).
