# 🌌 MidnightViolet

**System24 supercharged** — an inky dark theme with refined violet accents, designed for long Discord sessions.

Built on top of [System24](https://github.com/refact0r/system24) by refact0r, heavily customized with a deep violet palette, glow effects, custom animations, and dedicated syntax highlighting.

---

## ✨ Features

- 🎨 **Refined Violet Palette** — near-black backgrounds with carefully tuned violet accents
- 🔤 **JetBrains Mono + Space Grotesk** — professional monospace typography
- ✨ **Glow Effects** — borders that glow on hover across servers, inputs, and mentions
- 🎬 **Micro-Animations** — message entrance, popout transitions, blinking placeholder
- 🖥️ **Custom Syntax Highlighting** — neon colors for code blocks
- 📜 **Slim Scrollbar** — minimal neon-styled scrollbar
- 🔔 **Badge & Notifications** — glow effect on notification badges
- 🎵 **Custom Spotify Bar** — personalized playback bar
- ♿ **Accessibility Focus** — violet outline on focused elements

## 🚀 Installation on Vesktop

### Method 1 — Online Import (Recommended)

This method loads the theme directly from GitHub, so you'll automatically receive updates.

1. Open **Vesktop**
2. Go to **Settings** → **Vencord** → **Themes**
3. In the **Online Themes** section, paste this link into the text field:

```
https://raw.githubusercontent.com/L0orexist/MidnightVioletTheme/master/MidnightViolet.theme.css
```

4. Press **Enter** to confirm
5. The theme will be applied immediately! 🎉

---

### Method 2 — Local Installation (CSS File)

If you prefer to have the file locally:

1. **Download the file** [`MidnightViolet.theme.css`](https://github.com/L0orexist/MidnightVioletTheme/raw/master/MidnightViolet.theme.css)
   *(Right-click → "Save link as..." or download the repo ZIP)*
2. Open **Vesktop**
3. Go to **Settings** → **Vencord** → **Themes**
4. Click on **Open Themes Folder** (📁)
5. **Copy** the `MidnightViolet.theme.css` file into the folder that opened
6. Go back to Vesktop — the theme will appear in the list
7. **Enable it** with the toggle ✅

> [!TIP]
> With the local method, you'll need to re-download the file to get updates.

## 🔧 Customization

You can easily tweak colors and options by editing the CSS variables:

```css
/* Change the main font */
--font: 'JetBrains Mono';

/* Change the primary accent color */
--accent-1: oklch(72% 0.13 295);

/* Adjust background brightness */
--bg-4: oklch(9% 0.008 290);
```

For a full list of configurable variables, check the [System24 documentation](https://github.com/refact0r/system24).
